### 项目安装
```
npm install

或

cnpm install
```

### 项目运行
```
npm run serve-dev

或

npm run serve-prod
```

### 编译项目
```
npm run build-dev

或

npm run build-prod
```

### 目录结构
```
pear-admin-ant
|-src
  |-api             http请求
  |-assets          静态资源
  |-component       常用组件目录       
  |-config          项目配置文件
  |-directives      自定义指令目录
  |-layout          页面整体布局-layout
  |-mock            mock数据
  |-router          页面路由 
  |-utils           工具类
  |-view            页面目录
  |-App.vue         Vue主文件
  |-main.js         Vue加载启动文件
```
